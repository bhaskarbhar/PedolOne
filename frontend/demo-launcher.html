Setup Instructions

1. Start Backend Server

cd PedolOne/backend
python main.py

Backend will run on http://localhost:8000

2. Start Demo Apps

cd frontend/bank-app
npm run dev
cd frontend/insurance-app
npm run dev

Access URLs:

  • • StockBroker: http://localhost:3000
  • • BankABC: http://localhost:3001
  • • InsuranceABC: http://localhost:3002
